CSS Measuring Units

Length in CSS can be definied as a number followed by a length unit. For example, 10px or 2em.

There are two different types of lengths: relative and absolute.

Absolute Lengths

These are units that are fixed and appear exactly as the stated size.

  • cm - centimeters
  • mm - millimeters
  • in - inches (1in = 2.54cm)
  • px - pixels (1px = 1/96th of 1in)
  • pt - points (1pt = 1/72th of 1in)
  • pc - picas (1pc = 12pt)

Relative Lengths

These are units that specify a length in relation to another length property.

  • em - relative to the font-size of the element (2em means 2 times the size of the current font)
  • ex - relative to the x-height of the current font (rarely used)
  • ch - relative to the width of the "0" (zero) character in the current font
  • rem - relative to the font-size of the root element (html)
  • vw - relative to 1% of the width of the viewport
  • vh - relative to 1% of the height of the viewport
  • vmin - relative to 1% of the viewport's smaller dimension
  • vmax - relative to 1% of the viewport's larger dimension
