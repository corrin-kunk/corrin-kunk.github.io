CSS Measuring Units

Length in CSS can be definied as a number followed by a length unit. For example, 10px or 2em.

There are two different types of lengths: absolute and relative. Absolute units are meant to be fixed measurements, while relative units change depending on things like font size or viewport size

Absolute Lengths

Absolute length units are most commonly used when you want consistent sizing that does not depend on screen size or user settingfs. They are especially useful for print layouts or small design details that should stay the same everywhere.

The most common absolute units you'll see in CSS include:

  • cm - centimeters
  • mm - millimeters
  • in - inches (1in = 2.54cm)
  • px - pixels (1px = 1/96th of 1in)
  • pt - points (1pt = 1/72th of 1in)
  • pc - picas (1pc = 12pt)

Relative Lengths

Relative length units are based on another measurement, such as the font size of an element or the size of the browser window. Because they can scale and adjust, they are commonly responsive web design and help layouts adapt to different devices and user settings.

Some common relative units include:

  • em - relative to the font-size of the element
  • ex - relative to the x-height of the current font (rarely used)
  • ch - relative to the width of the "0" (zero) character in the current font
  • rem - relative to the font-size of the root element (html)
  • vw - relative to 1% of the width of the viewport
  • vh - relative to 1% of the height of the viewport
  • vmin - relative to 1% of the viewport's smaller dimension
  • vmax - relative to 1% of the viewport's larger dimension

What is em and rem?

em is a relative unit that is based on the font-size of the element it is used on. For example, if an element has a font-size of 16px, then 1em would be equal to 16px. If you set the font-size of an element to 2em, it would be 32px (2 times the font-size of the element).

rem is similar to em, but it is relative to the font-size of the root element (html) instead of the current element. This means that if the root element has a font-size of 16px, then 1rem would always be equal to 16px, regardless of the font-size of the current element.

What is vw and vh?

vw and vh are relative units that are based on the size of the viewport. vw is relative to the width of the viewport, while vh is relative to the height of the viewport. For example, if the viewport is 1000px wide, then 1vw would be equal to 10px (1% of 1000px). If the viewport is 800px tall, then 1vh would be equal to 8px (1% of 800px).

What is vmax and vmin?

vmax and vmin are relative units that are based on the size of the viewport. vmax is relative to the larger dimension of the viewport, while vmin is relative to the smaller dimension of the viewport. For example, if the viewport is 1000px wide and 800px tall, then 1vmax would be equal to 10px (1% of 1000px) and 1vmin would be equal to 8px (1% of 800px).
