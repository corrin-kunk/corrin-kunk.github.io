CSS Color

What do these CSS properties do?

Color

The color property sets the text color for an HTML element, such as a paragraph, heading, or link.

Background Color

The background-color property changes the background color behind an element, such as a paragraph, div, or the whole page.

Border Color

The border-color property controls the color of the border around an element. It is usually used with the border-style and border-width properties.

Named Colors

Named colors are predefined color names in CSS that you can use instead of writing out color codes.

Hexadecimal colors

Hex colors use a # followed by six characters that represent red, green, and blue values.

Hexadecimal colors with trasparency

Hex color can also include two extra characters at the end that control the transparency of the color.

RGB colors

RGB colors define color using red, green, and blue values that range from 0-255.

RGBA colors

RGBA works the same as RGB but adds an extra value that controls the transparency.

HSL colors

HSL defines color using hue, saturation, and lightness instead of numbers for red, green, and blue.

HSLA colors

HSLA works like HSL but includes an alpha value that controls transparency.

Current Color Keyword

The currentcolor keyword uses the element's current text color as the value for another property.
