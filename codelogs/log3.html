CSS Media Queries

How do they work?

CSS media queries are a powerful tool that allows developers to create responsive web designs. They work by applying different styles based on the characteristics of the device or screen size. This means that you can tailor the appearance of your website to look great on desktops, tablets, and smartphones without having to create separate versions of your site.

Why are they useful?

Media queries are useful because they enhance the user experience by ensuring that your website is accessible and visually appealing across a wide range of devices. They allow you to optimize the layout, typography, and overall design for different screen sizes, which can lead to increased engagement and satisfaction for your users.

Media Features Supported by Media Queries

Orientation

Detects whether the screen (viewport) is in portrait or landscape mode. 


            @media screen and (orientation: landscape) {
                main {
                    background-color: lightblue;
                }
            }

Min-Width

Applies styles when the viewport is at or wider than a specified width. 


            @media (min-width: 900px) {
                body {
                    font-size: 1.1em;
                }
            }

Max-Width

Applies styles when the viewport is at or narrower than a specified width. 


            @media (max-width: 600px) {
                main {
                    padding: 1em;
                }
            }

Resolution

Checks the pixel density of a display. 


            @media (min-resolution: 2dppx) {
                img {
                    image-rendering: auto;
                }
            }
