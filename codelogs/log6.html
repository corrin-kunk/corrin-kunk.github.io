CSS Property:

grid-template-columns

This property controls how many columns are in a grid and how wide they are. I used it in my site to create layouts like 3-column category buttons and 2-column featured article card sections.

Example:


            .category-grid {
                grid-template-columns: repeat(3, 1fr);
            }

In this example, the repeat(3, 1fr) function creates 3 equal-width columns in the grid.
