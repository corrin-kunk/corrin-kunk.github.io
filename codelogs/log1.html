Code Log 1: HTML Semantic Tags

Semanatic tags in HTML clearly describes its meaning to both the browser and the developer, whereas non-sematic tags tells nothing about its content.

Benefits of using semenatic tags:

  • Improved accessibility for screen readers and assistive technologies (AT)
  • Better SEO (Search Engine Optimization)
  • Makes websites more compatible across different browsers and devices
  • Helps code stay compatible with future technologies
