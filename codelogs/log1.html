HTML Semantic Tags

In HTML, Semanatic Tags clearly describe the meaning of the content they wrap to both the browser and the developer. Tags like <header>, <article>, <footer> are examples of semantic tags.

Non-Sematic Tags= tell nothing about their content.

Examples of Non-Sematic Tags

Benefits of using semenatic tags:

  • Improve accessibility for screen readers and assistive technologies (AT)
  • Better SEO (Search Engine Optimization)
  • Makes websites more compatible across different browsers and devices
  • Helps code stay compatible with future technologies
