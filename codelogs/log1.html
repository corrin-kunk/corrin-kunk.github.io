HTML Semantic Tags

In HTML, Semanatic Tags clearly describe what content is and how it is meant to function on a webpage. Semantic tags like <header>, <article>, and <footer> help define the overall structure of a webpage, not just the layout.

Benefits of using semenatic tags:

  • Improve accessibility for screen readers and assistive technologies (AT)
  • Better SEO (Search Engine Optimization)
  • Make websites more compatible across different browsers and devices
  • Help code stay compatible with future technologies

Whereas, Non-Sematic Tags such as <div> <span> do not provide any information about the content they contain. They are primarily used for styling and layout purposes. They require extra context, like class names or CSS, to explain their purpose.
